# Post-Quantum Notary Scheme for Blockchain Interoperability (PQ-NS-IOP)

**PQ-NS-IOP** is a blockchain interoperability bridge simulation based on the Notary Scheme protocol, implemented in **Go**.  

This research project analyzes the impact of **post-quantum cryptography** on the performance of interoperability bridges.  
We developed and evaluated a **novel dynamic trust model** among notaries within the Notary Scheme approach.  

For details on the implementation and architecture, please refer to my thesis:  
[Designing a Decentralized Quantum-Resistant Notary Scheme for Blockchain Interoperability with a Dynamic Trust Model](https://mspace.lib.umanitoba.ca/server/api/core/bitstreams/decea942-1b50-467c-a8b3-9f0afb80b4f1/content).  

This code was developed in the **[TCDT Lab](https://tcdt.ca)** at the University of Manitoba.
